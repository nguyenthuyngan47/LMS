# Changelog

## [1.0.0] - 2026-01-19

### Added
- Đăng nhập học viên với Odoo authentication
- Xem danh sách khóa học đã đăng ký
- Xem lộ trình học tập được gợi ý (Roadmap)
- Theo dõi tiến độ học tập theo từng khóa học
- Profile screen với thông tin học viên
- Home screen với quick actions và statistics
- Navigation bar với 4 tab chính

### Technical
- Flutter 3.0+ support
- Provider state management
- GoRouter for navigation
- Odoo JSON-RPC API integration
- Material Design 3 UI
